# Layout Utils

## Wrap

```css
..bb-{key}
```

<table>
<tr>
<th>key</th>
<th>description</th>
</tr>
<tr>
<td>wrap</td>
<td>flex-wrap: wrap</td>
</tr>
<tr>
<td>no-wrap</td>
<td>flex-wrap: nowrap</td>
</tr>
</table>
